HTML y CSS para principiantes

HTML y CSS son dos herramientas importantes para dar inicio en el mundo del desarrollo web. Por lo tanto, en este curso aprenderás todo sobre HTML y CSS, desde lo más básico hasta un nivel intermedio, logrando desarrollar tu primera página web responsiva, sin la necesidad de aplicar librerias externas para CSS.

Estilos con SASS

SASS es una herramienta importante hoy en día para todo desarrollador web, ya que permite crear estilos más ordenados, eficientes y con menos líneas de codigo. Por ello, en este curso aprenderás todo lo necesario para crear hojas de estilo desde la sintaxis de SCSS, implementando técnicas propias de SASS.
